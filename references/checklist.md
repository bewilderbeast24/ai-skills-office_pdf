# Checklist: pdf-processing Execution

## Stage 1: Mode Selection
- [ ] Identify the user's specific PDF processing requirement and select the appropriate operational mode.

## Stage 2: Context Setup
- [ ] Load relevant code snippets, templates, and library configurations based on the selected mode.
- [ ] Verify required libraries and CLI tools are available.

## Stage 3: Specialized Workflow
- [ ] Execute the specific PDF operation (Basic, Extraction, Creation, Form Filling, Advanced) using gathered context.

## Stage 4: Outcome
- [ ] Validate final output to ensure it meets requirements.
- [ ] Present result to the user and perform cleanup of temporary files.
